# MVP — Sistema de Controle de Despesas (v0.0.3)

## Funcionalidades Entregues
- Cadastro de despesas com gravação em `data/despesas.txt`
- Cadastro de pagamentos com gravação em `data/pagamentos.txt`
- Cadastro de usuários com senha criptografada (SHA-256)
- Listagem de despesas
- Menu principal funcional
